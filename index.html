Project 2
Rogue DHCP Server Discovery Tool

Built a network application using python scapy module which detects rogue DHCP servers on each local area network that is connected to the Ubuntu VM. Created a DHCP discover packet and used scapy to send the packet to all the connected LANs and wait for the reply to detect both legit and rogue DHCP servers.

Project 3
Filter and Monitor LLDP Log Messages

Built a network application that reads a bunch of IP addresses from a file, connects to each of these addresses over SSHv2 using the netmiko module and different sets of login credentials. Then it sends a command that shows all the log messages that each network device generated over the past day. Then it saves the output from each device and finally extracts only the log messages related to LLDP Neighborships. Then it generates a text report containing the filtered LLDP log messages.

Project 4
Subnet Calculator

Built a network application which is a simple command line subnet calculator which takes IP address and subnet mask as input and returns five parameters as output - Network Address, Broadcast address, Number of valid hosts per subnet, Wildcard mask, Number of mask bits. Finally, the application generates a Random IP address from the input subnet in which the initial IP address resides.
